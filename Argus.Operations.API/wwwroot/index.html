ARGUS Operations API

Central operacional de combate a incêndios florestais

API .NET responsável pela gestão de brigadas, brigadistas, ocorrências em campo, recursos materiais e registros de atendimento. Autenticação JWT, autorização granular por perfil e proxy de leitura para a API Intelligence Java.

Visão executiva

ARGUS Operations API

Camada operacional do ecossistema ARGUS, responsável por receber os alertas gerados pela inteligência ambiental e coordenar a resposta em campo: despacho de brigadas, abertura de ocorrências, registros de atendimento e gestão de recursos.

Construída sobre .NET 9 e ASP.NET Core com Clean Architecture, persiste no Oracle via EF Core Code First, autentica com JWT/BCrypt, publica logs estruturados via Serilog e consome a API Java por HTTP tipado para alertas e focos de calor.

Domínios de negócio ARGUS

Brigadas

Equipes de combate, com nome, base operacional e contato. Âncora operacional dos brigadistas e recursos.

Brigadistas

Membros operacionais vinculados a uma brigada, com matrícula, função e dados de contato.

Ocorrências

Registros de combate em campo, opcionalmente vinculados a um alerta da API Java pelo campo AlertaId.

Recursos

Equipamentos operacionais (caminhão-pipa, EPI, ferramenta) alocados por brigada.

Alertas (proxy Java)

Proxy de leitura para os alertas gerados pela API Intelligence. Permite promover um alerta a ocorrência.

Focos de Calor (proxy)

Proxy de leitura para os focos brutos do satélite (NASA FIRMS) servidos pela API Java.
